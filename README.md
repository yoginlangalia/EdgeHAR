# EdgeHAR

EdgeHAR is a tool designed to facilitate the analysis of network requests and responses during the use of web applications. It gathers HAR (HTTP Archive) data, which is useful for debugging performance issues, understanding loading times, and identifying bottlenecks in web applications.

## Features

- Capture network requests and responses
- Analyze performance timings
- Export HAR files for further analysis
- User-friendly interface for visualizing data

## Installation

To install EdgeHAR, follow the steps below:

1. Clone the repository:
   ```bash
   git clone https://github.com/yoginlangalia/EdgeHAR.git
   ```
2. Navigate to the project directory:
   ```bash
   cd EdgeHAR
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage

To start using EdgeHAR, run the following command:
```bash
npm start
```

Visit `http://localhost:3000` in your browser to access the application.
